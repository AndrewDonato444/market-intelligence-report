# IntelligencePillars

**Status**: Stub (pending implementation)

## Purpose
Three-column layout presenting core value propositions. Each pillar opens with a small gold accent line (not an icon), followed by a serif title and concise body copy. Generous white space. Warm background.

## Props
- `pillars`: Array<{ title: string, description: string }>

## Design Tokens
- Background: `color-surface`
- Accent mark: `color-accent`, 2px height, 24px wide
- Title: `font-serif`, `text-xl`, `color-primary`
- Description: `font-sans`, `text-sm`, `color-text-secondary`, line-height 1.7
- Spacing: `spacing-8` gap between pillars, `spacing-4` internal

## Responsive
- Desktop: 3 columns
- Tablet: 3 columns with reduced spacing
- Mobile: single column, stacked
